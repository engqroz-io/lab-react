## Functions

### Map()
Map() is a loop function that allow apply an operation over each item.

```javascript
let numbers = [1, 2, 3, 4, 5]

// multiply each value in array by 2
numbers = numbers.map(n => n * 2)
console.log(numbers)
```

## Operators

- Use conditional operator "==" if you want to valid if one value is equal to another, regardless of type.
- Use conditional operator "===" if you also want to check the type.

> follow the same logic for other conditional operators.
