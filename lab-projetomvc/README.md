# Projeto MVC - Agenda de Contatos

> # MVC Project - Contact Agenda

Olá! Este é meu projeto MVC em .NET para gerenciar uma agenda de contatos. Usei ASP.NET Core com Entity Framework para o banco de dados.

> Hello! This is my MVC project in .NET to manage a contact agenda. I used ASP.NET Core with Entity Framework for the database.

## 🛠️ Stacks Utilizadas / Used Stacks

- **C#**: Linguagem do backend. `Backend language.`
- **ASP.NET Core MVC**: Para a estrutura web. `For the web structure.`
- **Entity Framework**: Para acessar o banco de dados. `To access the database.`
- **SQL Server** (ou similar, via migrations): Para armazenar os contatos. `To store the contacts.`
- **Razor Views**: Para as páginas HTML. `For the HTML pages.`

Foi legal aprender sobre MVC e como conectar tudo com o banco! 📅

> It was cool to learn about MVC and how to connect everything with the database! 📅
